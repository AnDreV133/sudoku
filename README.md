# sudoku
Простая реализация игры в Судоку.
===========

##### Управление:
 * Клавиатура:
     * Клавиша пробел: новая игра
 * Мышка:
     * Левая кнопка: +1 к текущему числу
     * Правая кнопка: -1 к текущему числу
     * Средняя кнопка: очистить значение
